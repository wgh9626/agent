# 状态枚举

## 最终状态（final status）

- `success`：发布成功
- `failed`：发布失败
- `rolled_back`：发布失败且已执行回滚

---

## 执行阶段（stage reached）

- `init`：初始化
- `checkout`：检出代码
- `build`：Maven 构建
- `image`：镜像构建
- `push`：镜像推送
- `deploy`：部署执行
- `verify`：发布验证

---

## 步骤结果（step result）

- `success`：成功
- `failed`：失败
- `skipped`：跳过
- `not_started`：未开始

---

## 健康检查结果（health result）

- `UP`：健康
- `DOWN`：不健康
- `UNKNOWN`：未知
- `NOT_CHECKED`：未检查
